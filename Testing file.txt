Open Linux with a username default

clone the repo https://github.com/chirumama/mycrudapp

install dotnet-runtime-8.0
by using these command

sudo apt-get-repository ppa:dotnet/backport -y
sudo apt update
sudo apt install -y dotnet-sdk-8.0

install posgres and add username and password and pass preveilge access

Create database itemdb
Add table id,title,description,image_url,link

update the same password&username in mycrudapp/backenddot/appsetting.json 


{
  "ConnectionStrings": {
    "DefaultConnection": "Host=localhost;Port=5432;Database=itemdb;Username=;Password="
  },
  "Logging": {
    "LogLevel": {
      "Default": "Information",
      "Microsoft.AspNetCore": "Warning"
    }
  },
  "AllowedHosts": "*"
}

install nginx

allow full permission to nginx

sudo chmod 755 to /home/username
sudo chmod -R 755 /home/username/mycrudapp

change made on file 
/etc/nginx/sites-available/default

server {
    listen 80;
    server_name localhost;

    root /home/username/mycrudapp/frontend;
    index index.html;

    location / {
        try_files $uri $uri/ =404;
    }

    # 2. Reverse proxy to your .NET backend API
    location /api/ {
        proxy_pass http://localhost:5000;         proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection keep-alive;
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}


save this and then restart the ngnix

start dotnet username/mycrudapp/backenddot/CRUDapp.dll --urls=http://localhost:5000

show IP to redirect to browser
